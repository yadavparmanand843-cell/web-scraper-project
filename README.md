# Web Scraper Project

This is a Python-based web scraping project that extracts quotes and author names from a website and stores them into a CSV file.

## Technologies Used
- Python
- BeautifulSoup
- Requests
- Pandas

## Features
- Automatic quote extraction
- CSV file generation
- Simple data automation## Files- scraper.py → main Python script- quotes.csv → extracted data## Author* parmanand yadav
